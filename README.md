# OptiCrop: Smart Agricultural Production Optimization Engine

OptiCrop is an intelligent, data-driven agricultural recommendation system that analyzes soil and environmental factors to recommend the most suitable crops for farmers. By leveraging machine learning models, OptiCrop aims to maximize crop yield, enhance resource efficiency, and support sustainable agricultural practices.

---

## 📁 Repository Structure

This repository is organized into milestones tracking the project lifecycle from ideation to delivery:

*   **`1.Brainstroming/`**: Problem statements, empathy mapping, and prioritization matrix.
*   **`2/`**: Technology stack selection, customer journey mapping, solution requirements, and data flow diagrams.
*   **`3/`**: Problem-solution fit, proposed solution overview, and system architecture.
*   **`4/`**: Project planning and scheduling sheets.
*   **`5/`**: Coding & implementation solutions, codebase layout, and reusability reports.
*   **`6/`**: Performance testing and analysis reports.
*   **`7/`**: Project executable source code (Flask web application in `7/crop/crop/`).
*   **`8/`**: Demo planning, team involvement records, future scalability plans, and demonstration videos.

---

## 🛠️ Technology Stack

*   **Backend Framework**: Python (Flask)
*   **Machine Learning**: scikit-learn (Logistic Regression, K-Means clustering)
*   **Data Processing**: Pandas, NumPy
*   **Frontend**: HTML5, CSS3, Google Fonts (Outfit)

## 🚀 How to Run the Application Locally

### Prerequisites
Make sure you have Python installed, then install the required libraries:
```bash
pip install flask scikit-learn pandas numpy jupyter
```

### Running the App
1. **Navigate to the web application directory**:
   ```bash
   cd "7/crop/crop/data_collection_analysis"
   ```
2. **Start the Flask server**:
   ```bash
   python app.py
   ```
3. **Access the application**:
   Open your web browser and go to `http://127.0.0.1:5000/`
